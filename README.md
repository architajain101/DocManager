# 📂 DocManager

DocManager is a document management system built using Python, Streamlit, and database integration. It helps users organize, store, and manage documents efficiently with tagging, search, analytics, and status tracking features.

---

# 🚀 Features

✅ Document upload and storage
✅ Document tagging system
✅ Smart search functionality
✅ Status tracking (Completed / Pending)
✅ Organized document storage system
✅ PDF management and metadata handling
✅ Interactive Streamlit dashboard
✅ Database integration for structured storage
✅ Analytics and tracking support

---

# 🛠️ Tech Stack

* Python 🐍
* Streamlit 📊
* SQLite / MySQL 🗄️
* Pandas
* Object-Oriented Programming (OOP)

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/architajain101/DocManager.git
```

Move into the project directory:

```bash
cd Docmanager
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app/main.py
```
